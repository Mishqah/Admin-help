Search admin

1. Navigate to the admin screen by clicking the manage staff card, which will be positioned in the body of the admin home screen.

2. Once on the manage staff screen, the current admin must click the admin card which will be on the left side of the screen.

3. System will navigate current admin to the admin screen.

4.Â  Navigate to the search bar input field, which is positioned above the admin table, and enter the desired admin details. User can search by name, surname, email address or contact number. When user enters details into search bar, the admin table will be filtered out to return the users desired results and display them.
