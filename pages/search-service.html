Search Service

When accessing the Service part of the system the admin will need to be logged in and navigate to the admin home screen. In the admin home screen, there will be numerous buttons containing links to the various subsystems. The admin will have to click on the services button to gain access to the Services Type screen. Once on the Service Type Screen The admin will click on Services Tab on the left to gain access to the Services Screen.

  

When the desired information is searched in the input field the table will filter out according to the criteria entered.
